## 资源内容

- **文件名**: `KnifeEdgeMTF.m`

- **功能**: 通过刀刃法计算图像的MTF曲线

## 使用说明

1. **环境要求**: 确保你的计算机上已安装MATLAB软件。
2. **文件导入**: 将`KnifeEdgeMTF.m`文件导入到你的MATLAB工作目录中。
3. **运行脚本**: 在MATLAB命令窗口中运行该脚本，输入所需的图像数据。
